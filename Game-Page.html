2D Adventure Game

Players usually control a character moving through various environments, interacting with objects, solving puzzles, and talking to non-playable characters to progress through the story. 2D sprites are used to represent characters and objects, giving a classic pixelated aesthetic.

Tic-Tac-Toe Game

The game is played on a grid that's 3 squares by 3 squares. You are X , your friend (or the computer in this case) is O . Players take turns putting their marks in empty squares. The first player to get 3 of her marks in a row (up, down, across, or diagonally) is the winner.

Hangman Game

Hangman is a guessing game for two or more players. One player thinks of a word, phrase, or sentence and the other(s) tries to guess it by suggesting letters or numbers within a certain number of guesses. Originally a paper-and-pencil game, there are now electronic versions.

Memory Game

The memory game is a common children's game played with a set of cards. The cards have a picture on one side and each picture appears on two (or sometimes four) cards. The game starts with all the cards face down and players take turns to turn over two cards.
